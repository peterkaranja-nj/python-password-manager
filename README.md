**Python Password Manager (CLI)**



A simple command-line password manager written in Python.



Features



- Add passwords
- Retrieve passwords
- Delete passwords
- Show stored services
- Count stored passwords
- Generate random passwords
- Search for stored services
- Logging of actions
- Persistent storage using a file




Technologies Used


- Python
- File handling
- Dictionaries
- CLI interaction
- Random & string modules for password generation
- Logging system




How to Run



bash

python password\_manager.py
